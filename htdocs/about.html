CLASS NETWORK It is a platform for the students of the class to connect with each other and get information about the current happenings of class. The website will include the following

  • Student’s profile
  • Time table
  • Submissions
  • Events
  • Reference material
  • Daily updates

Scope of Project Class network website allows students to view and edit information including schedule, class activities, reference material etc. It is a platform for the students of the class to connect with each other and get information about the current happenings of class. In his profile a student can view the schedule, class activities, reference material etc. A student can also get his doubts cleared online via an online chat box which directly connects students .

Design and implementation constraint To build Class network and put it online requires knowledge of Web Programming. In Web Programming we have HTML, HTML 5.0 – To design the basic structure of Website System CSS – To give various Styles to Website System PHP, JAVASCRIPT – To make a dynamic and user interactive System. And Server side and Client side programming. Also System has Database connectivity using standard Database Servers like Oracle,Access (Jet, MSDE), MySQL, SQLBase, SQL Server, Teradata, SUPRA, IBM DB2 etc.
